# Vibe Commerce Mock E-Commerce Cart

## Overview
This is a basic full-stack shopping cart application built for Vibe Commerce screening.  
It allows users to browse products, add/remove items from the cart, and perform a mock checkout.

## Tech Stack
- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **API:** REST APIs

## Features
- Display products grid with "Add to Cart" button
- View cart with items, quantities, and total price
- Update item quantity and remove items from cart
- Checkout form with name/email/address
- Receipt modal showing purchased items and total
- Responsive design
